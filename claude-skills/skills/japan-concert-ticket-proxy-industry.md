# 日本演唱會門票代行產業分析

## 市場概況

| 項目 | 數據 |
|------|------|
| **日本現場娛樂市場** | 7,605 億日圓（2024） |
| **演唱會門票年銷售量** | 約 5,000 萬張 |
| **抽選制普及率** | 主要票務平台 80%+ 採用抽選制 |
| **代行服務歷史** | 30+ 年（F&C 成立於 1995 年） |

## 法律框架

### チケット不正転売禁止法（2019 年施行）

| 行為 | 合法性 |
|------|--------|
| **代行申込**（代替抽選） | **合法** — 代替他人參加抽選 |
| **転売目的の購入** | **違法** — 以轉賣為目的購票 |
| **定価超の転売** | **違法** — 超過定價轉賣 |
| **定価以下の譲渡** | **合法** — 定價或以下轉讓 |

**關鍵：代行 ≠ 転売。代行是合法的「代替申込」服務。**

## 主要票務平台（抽選入口）

| 平台 | 市佔 | 特色 |
|------|------|------|
| **ぴあ（PIA）** | 最大 | 會員數最多，幾乎所有公演 |
| **ローソンチケット** | 第二 | Loppi 便利商店取票 |
| **e+（イープラス）** | 第三 | 電子票增長中 |
| **Ticketmaster Japan** | 海外系 | 國際公演、K-POP 強 |
| **各 FC 先行** | — | Fan Club 會員優先抽選 |

## 競品分析：前三大代行服務

### 1. F&C（f-and-c.com）— 會員制老牌

| 項目 | 詳情 |
|------|------|
| **成立** | 1995 年（31 年） |
| **模式** | **會員月費制** + 申込手數料 |
| **月費** | ¥3,000 ~ ¥50,000/月（依等級） |
| **入會費** | ¥5,000 ~ ¥7,000 |
| **手數料** | ¥5,000 ~ ¥7,000/件 |
| **退會** | 需提前一個月通知 |
| **網站技術** | 傳統 HTML，非響應式 |
| **強項** | FC 先行管道多、歷史信譽 |
| **弱點** | 月費門檻高、網站老舊、退會流程不透明 |

### 2. チケット119（ticket-119.com）— 成功報酬制

| 項目 | 詳情 |
|------|------|
| **成立** | 1998 年（28 年） |
| **模式** | **無會員制**，成功才收費 |
| **手數料** | ¥4,000/張（最低消費） |
| **會員費** | 無 |
| **網站技術** | 基礎 HTML，有手機版 |
| **強項** | 零風險（抽不到不收錢）、低門檻 |
| **弱點** | 熱門公演成功率低、單價較高 |

### 3. チケットジャム — 仲介型

| 項目 | 詳情 |
|------|------|
| **模式** | 仲介平台（買方 + 賣方） |
| **手數料** | 成交價的 10-15% |
| **特色** | 電子票對應、即時交易 |

## 競爭者網站共通弱點

1. **UI/UX 老舊** — 多數仍是 2000 年代設計
2. **無 AI 整合** — 沒有智能推薦或自動化
3. **行程規劃缺失** — 只管票，不管旅程
4. **多語言缺乏** — 幾乎全日文，外國粉絲難用
5. **透明度不足** — 抽選成功率不公開

## 差異化機會

| 方向 | 說明 |
|------|------|
| **AI + 行程規劃** | 不只搶票，還規劃交通、住宿、周邊行程 |
| **多語言** | 中文/英文界面 → 海外粉絲市場 |
| **透明度** | 公開歷史成功率數據 |
| **現代化 UX** | 手機優先、即時通知、LINE 整合 |
| **訂閱 + 成功報酬混合** | 低月費 + 成功才收主要費用 |

## Notes

- 業主說「收費方式我有想法」— 定價策略由業主決定，不需要 AI 建議
- 代行服務的法律灰色地帶：需要確認是否需要「古物商許可」
- Fan Club 會員費是額外成本（每個藝人 FC 約 ¥3,000-5,000/年）
- 熱門公演（如嵐、King & Prince）的抽選倍率可達 50-100 倍
